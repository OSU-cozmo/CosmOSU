Installation
============

    Clone the repository_.

    .. _repository : https://github.com/OSU-cozmo/CosmOSU

    Then in the root directory run:

    ``pip install -e .``

Hello World
=============

    .. code-block:: python


        import CozmOSU as Robot

        robot = Robot.Robot();

        def main(cozmo : Robot.Robot):

            cozmo.say("Hello World");

        robot.start(main);
